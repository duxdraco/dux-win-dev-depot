# Screenshots & assets

Drop PNG screenshots here and reference them from the main
[README](../../README.md#screenshots). Suggested captures (Windows Terminal, dark theme):

| File | Command to capture |
|------|--------------------|
| `analyze.png` | `pwsh -File .\DevDepot.ps1 analyze` (grouped output) |
| `install.png` | `pwsh -File .\DevDepot.ps1 install --provider npm` (verification block) |
| `status.png`  | `pwsh -File .\DevDepot.ps1 status` |

Guidelines:

- Keep terminal width ~100 columns so lines don't wrap.
- Use a clean prompt (hide personal paths/usernames where practical).
- Optimize PNGs before committing.

A social-preview banner (1280×640) can also live here as `social-preview.png` and be set
under **Settings → General → Social preview**.
